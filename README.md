Mortal Kombat tower meme for the current dollar price in Argentina

## Install

```
$ npm install
```

## Config

Copy the `env.sample` to `.env`

```
$ cp env.sample .env
```

If you need a different output HTML location, change the `OUTPUT_HTML` env variable

## Run

```
$ npm run update
```
